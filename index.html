Social distancing and mask detection using deep learning

Project Brief

Artificial intelligence and computer vision are the emerging technologies and acts as the best solution to any real time problem. In this proposal the aim is to build a deep learning model using computer vision which mainly acts as solution to COVID-19 crisis. In this proposal, developing a system using deep learning frameworks such as Tensor flow, keras or Pytorch and Mobilenetv2 architecture used for fine tuning purpose using transfer learning and by applying these to algorithms such as CNN, faster RCNN or YOLO which acts as a social distancing tool and face mask detector to ensure users safety. This is also useful in providing security to people in public places by detecting whether a perso n is wearing face mask or not and to check if the area is crowded or not in order to avoid spread of any contagious disease. This gives effective and high accurate results.

Project Info

  • Date - 2021
  • Tools - Computer vision, Transfer learning, Keras/Tensor flow, CNN/faster RCNN, YOLO.

Phishing Website

Project Brief

Phishing is one of the luring techniques used by phishing artist in the intention of exploiting the personal details of unsuspected users. Phishing website is a mock website that looks similar in appearance but different in destination. The unsuspected users post their data thinking that these websites come from trusted financial institutions. Several antiphishing techniques emerge continuously but phishers come with new technique by breaking all the antiphishing mechanisms. Hence there is a need for efficient mechanism for the prediction of phishing website. This project employs Machine-learning technique for modelling the prediction task and supervised learning algorithms namely Decision tree induction, Naïve bayes classification and Random Forest are used for exploring the results.

Project Info

  • Date - 2020
  • Tools - Machine Learning, Random Forest, Naive Bayes decision tree, HTML, CSS, Python

Soldier Safety System

Project Brief

The basic challenge for any country is to maintain good security of its people, Army place very important role in fulfilling this goal. Their security becomes equally important. The purpose of this system is to help soldiers by keeping track of their health status and also by detecting any sign of enemies along their path. This system helps the necessary officers to keep track of any emergency. This is done by using GPS module that can provide real time location of soldiers which is sent via GSM to the control room. If any emergency arises the control room can send necessary help to the soldiers to keep them safe. In order to know the health status various sensors such as temperature sensor, heartbeat sensor, accelerometer is used. Heartbeat sensor can provide help by giving readings, in case of unusual heart rate control room will be able to take necessary action. Accelerometer defines fitness of soldiers. This system uses Image processing which acts as best tool in recognizing objects and making sure those soldiers are not under any threat.

Project Info

  • Date - 2020
  • Tools - GPS, GSM, Temperature sensor, Heartbeat sensor, Accelerometer

Temperature Control System

Project Brief

A Temperature Controlled System is a type of control system that automatically controls the temperature of an object or an area. We commonly use temperature control systems in Air Conditioners, Refrigerators, geysers, etc. where the temperature is automatically adjusted as per the input settings. In order to implement a temperature control system, we need a temperature sensor, a controller, and a cooling system. In this project, we have implemented a simple Temperature control system using simple components. The aim of this project is to automatically turn on or off the fan by detecting the surrounding temperature.

Project Info

  • Date - 2019
  • Tools - Temperature Sensor, LM358, L293D, Fan
